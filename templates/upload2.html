Instructions

Head over to the original source form (Do no send the original file!), make a copy and send the copy submission link to your customer.

When the form has been filled, in the responses tab, select Download responses and updload the file above and select the right url format

You'll get all the mappings pre-generated, ready to copy and paste in csapp, grouped by similar conversion structures (Less mapings to create)

Conversion types

Conversion types are using the goal name. IE, Goal2 will map to conversion_id goal2, Ecommerce to ecommerce

The above could create issues if 2 different profiles map to the same Goal ID but the goal is different between profile #1 and profile #3.IE:

Profile 1234 use Goal3 (New lead) and profile 4567 use Goal3 (Online sale). Either they'll get grouped under the same mapping (if their structure is identical), or at the very least under the same conversion_id

Also remember that GA conversion types are Conv date only
